Preguntas y respuestas:

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    2.
    Ambos son lenguajes de programación, pero la diferencia es que Java es compilado y JavaScript es interpretado. Quiere decir que Java pasa a través de un proceso para ser código de máquina y JavaScript se lee línea a línea en tiempo real mientras funciona el navegador. Amobos son distintos en su uso, declaraciones de datos y otras cosas más.
